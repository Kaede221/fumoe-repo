# @fumoe/taro-hooks

专为 Taro 4.x 设计的 React Hooks 集合，提供开箱即用的微信小程序和其他常用Hooks功能.
## 📦 安装

```bash
npm install @fumoe/taro-hooks
# 或者
yarn add @fumoe/taro-hooks
# 或者
pnpm add @fumoe/taro-hooks
```

## 🛠️ 开发

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm run dev

# 构建
pnpm run build

# 发布前检查
pnpm run prepublishOnly
```
